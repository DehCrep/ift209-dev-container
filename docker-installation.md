# Configuration de *Docker Desktop*

Cette procédure vous aidera à installer *Docker Desktop*, qui vient avec tout ce qui est requis pour correctement utiliser le DevContainer de ce projet.

*Elle devrait fonctionner pour Windows, Linux et Mac.*

## Installer *Docker Desktop*

1. Installez *Docker Desktop* à partir du [site officiel](https://docs.docker.com/desktop/).
2. Suivez les instructions de l'installateur. Il n'y a pas de configuration spéciale à faire.

## Configurer *Visual Studio Code*
1. Installez l'extension suivante dans *Visual Studio Code*:
    - [Remote Containers](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers)
